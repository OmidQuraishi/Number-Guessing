# JavaScript Number Guessing Game

A fun and interactive Number Guessing Game built with JavaScript where the player tries to guess a randomly generated number within a limited number of attempts. This project showcases core JavaScript concepts along with simple game logic and user interaction.

## Features

Random number generation
User input handling
Feedback hints (too high / too low)
Limited number of attempts
Win/lose game states with messages
Option to restart the game

## Built With

HTML5
CSS3
JavaScript (Vanilla JS)

## What I Learned

Generating random numbers in JavaScript
Working with conditional logic
Handling user input and validation
Updating the UI dynamically based on game state
Managing game flow and state
Functions, conditionals
